# Drive Source Notes

The working process assumes that term-specific NERDC source files may also live in Google Drive.

When handing this pack to a new AI, also upload or connect the exact scheme file for the requested class/subject/term if it is not already inside this ZIP.

Example verified source in Drive:
- Mathematics - Primary 1 - First Term
- Topics include number recognition 1-100, counting forwards and backwards, place value, addition, subtraction, money, time, and revision/examination.

If a new chat cannot see Google Drive directly, export the needed PDF/TXT/XLSX files and add them to the chat.
