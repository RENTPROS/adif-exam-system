RULES FOR GENERATION WORKFLOW
1. Check local syllabus first under:
   source-materials/NERDC_Scheme_of_Work_2025_2026/Primary_School/{CLASS}/{TERM}/
2. If present:
   - mark as verified-local
   - generate from that syllabus only
3. If missing:
   - mark as missing-local
   - invoke discovery logic in SYLLABUS_DISCOVERY_PROMPT.txt
4. For discovery:
   - prioritize NERDC, FME, UBEC official sources only
   - verify class/subject/term fit
   - apply SUBJECT_NAME_MAPPINGS.md equivalences
5. If official source found:
   - save syllabus into correct folder
   - append entry to source-materials/NERDC_Scheme_of_Work_2025_2026/SYLLABUS_DISCOVERY_LOG.md
6. If not found:
   - stop generation
   - show clear warning: official syllabus not found; no exam generated

EXAM OUTPUT CONSTRAINTS
- 2-slide PowerPoint workflow
- Slide 1: full school header
- Slide 2: continuation only (no repeated header)
- Section A: 20 objective questions
- Section B: 5 fill-in-the-gap questions
- Section C: 5 theory questions, answer any 3, each with (a) and (b)
- No diagram bank; diagrams must appear inside relevant question
- Easy difficulty (about 2/5), child-friendly wording
